# Product Brief — 자꿍이 1.0

## 1. 제품 정의

자꿍이는 시험 합격을 목표로 사용자의 점수/기간/학습시간을 입력받아, 매일 해야 할 학습과 문제를 제시하고 오답을 반복시키는 개인 맞춤형 학습 서비스입니다.

## 2. 핵심 가치

1. 선택과 집중: 낮은 점수 과목 우선
2. 실행 가능성: 과부하 없는 하루 학습량
3. 반복 학습: 오답 기반 재출제
4. 가시성: 합격 확률/성장 추이 제공

## 3. 문제 정의

기존 학습 앱은 계획과 문제풀이가 분리되어 있어, 약점 보완 루프가 약합니다.
자꿍이는 계획-풀이-오답반복을 하나의 흐름으로 통합합니다.

## 4. 1.0 목표 지표

- 온보딩 완료율
- 일간 학습 계획 수행률
- 오답 재풀이율
- 2주 연속 학습 지속률

## 5. 1.0 범위 요약

### Must
- 온보딩 (시험/점수/시간/시험일)
- 오늘의 학습 플랜 자동 생성
- GPT 문제 생성 및 풀이
- 오답 저장/복습 큐
- 합격 확률 v0

### Won't (1.0 제외)
- Vector DB 기반 고급 추천
- 대규모 큐 처리(BullMQ)
- 다중 모델 라우팅 최적화
