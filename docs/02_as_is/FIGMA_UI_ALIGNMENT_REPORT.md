# Figma UI 정합성 보고서

## 1. 비교 기준

- Figma 원본: Personalized Study Coach App
- 실제 구현: Vite React 프론트 (`src/app/pages`)

## 2. 정합성 요약

| 영역 | 정합성 | 코멘트 |
|------|--------|--------|
| 메인 대시보드 | 높음 | 카드형 UI, 합격확률, 약점 영역 반영 |
| 분석 시각화 | 높음 | Radar/Line chart 구현 |
| 문제 풀이 | 중간 | 4지선다/해설 구현, 오답저장 버튼 부재 |
| 오답노트 | 낮음 | 전용 화면/리스트 미구현 |
| 설정 | 중간 | UI 구현, 실제 저장/동기화 미구현 |

## 3. 디자인 토큰/테마

- 라이트 테마 기본 적용
- 다크 토큰은 존재하나 동작 스위치 미연동

## 4. UI 관점 필수 보완

- 모바일 퍼스트 레이아웃 정리
- 오답노트 전용 화면 추가
- 온보딩(입력 플로우) 화면 추가
